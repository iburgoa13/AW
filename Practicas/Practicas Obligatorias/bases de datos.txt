1 --> definir datos
2 --> diseñar las consultas SQL
nos vamos a centrar en datos para la vista (siempre SELECT)
Nos devuelve un array de objetos y cada objeto tiene tantas propiedades del select sacadas (quizas usar as en sql para nombrar)
ese array devuelto del select es incomodo para su inserción.
3 --> adaptar esos datos del array a nuestros objetos, esto facilita el volcado de datos en vista

Ejemplo:
Vista(contiene tabla con usuarios del sistema que son mayores de edad)
de la bbdd sacamos de la tabla Usuario (id,nombre,apellido,edad) --> SELECT * FROM Usuario WHERE EDAD >=18;
1--> DEFINIR LOS datos
2--> LA CONSULTA (select...) ==> devuelve un array de usuarios mayores de edad (son objetos)[{nombre:---},{id...,nombre,apellido,edad]
3--> queremos un array de cadenas , donde cadena tiene nombre y apellido del usuario separados por ' '
usamos las funciones de orden superior (IMPORTAAAANTE)
let resFinal = [];
resFinal.map(elemento => resFinal.push(elemento.Nombre + " " + elemento.Apellido));
